# GithubFinder
一款基于React Native开发的项目，一份代码可运行在ios和android上。

![项目简介](https://github.com/byk04712/GithubFinder/blob/master/introduce/clip.gif)

---


- components 目录主要存放显示页面和公共组件
- logic 目录主要存放业务逻辑处理类
- utils 目录主要存放api接口调用和一些工具类
