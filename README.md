### 安装步骤

1.  `git clone git@github.com:byk04712/GithubFinder.git`

1.  `cd GithubFinder && npm install`

1.  `open ios/GithubFinder.xcodeproj`






---



### 目录说明
- app/components 目录主要存放显示页面和公共组件
- app/logic 目录主要存放业务逻辑处理类
- app/utils 目录主要存放api接口调用和一些工具类
